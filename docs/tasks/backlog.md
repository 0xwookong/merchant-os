## 依赖关系总结

```
Auth（认证）→ Layout（布局壳）→ Environment（环境切换）
    ↓                                    ↓
 注册/登录 ──→ KYB ──→ 入驻申请          │
    ↓                    ↓               ↓
 成员管理            仪表盘/订单    开发者控制台
                                     ↓
                              API 文档引擎
                              签名工具
                              Webhook
                              域名白名单
                              API 日志
                                  ↓
                              MCP 集成
```
