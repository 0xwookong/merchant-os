# 商户注册、登录与 KYB 认证

## 1. 功能概述

商户首次使用 OSLPay Merchant Portal 需要完成 **注册 → 登录 → KYB 引导认证** 三个环节。KYB（Know Your Business）认证是合规要求，商户完成认证后方可进行正式的支付业务。

## 2. 商户注册

### 2.1 注册入口
- 登录页面提供「注册」入口链接
- 支持邮箱注册

### 2.2 注册流程

```
用户访问注册页面
    ↓
填写注册信息（邮箱、密码、公司名称等）
    ↓
提交注册
    ↓
系统发送验证邮件
    ↓
用户点击邮件中的验证链接
    ↓
邮箱验证通过，注册完成
    ↓
自动跳转至登录页面
```

### 2.3 注册信息（预期字段）
| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 邮箱 | Email | ✅ | 作为登录账号，需邮箱验证 |
| 密码 | Password | ✅ | 需满足密码复杂度要求 |
| 确认密码 | Password | ✅ | 二次确认 |
| 公司名称 | Text | ✅ | 商户公司全称 |
| 联系人姓名 | Text | ✅ | 主联系人姓名 |

### 2.4 密码规则
- 最少 8 个字符
- 包含大写字母、小写字母、数字中的至少两种
- 不允许与邮箱相同

## 3. 商户登录

### 3.1 登录方式
- 邮箱 + 密码登录
- 后续可扩展：SSO、OAuth2.0、MFA（多因子认证）

### 3.2 登录流程

```
用户访问登录页面
    ↓
输入邮箱和密码
    ↓
提交登录
    ↓
[验证通过] → 检查 KYB 认证状态
    ├── [未完成 KYB] → 跳转至 KYB 引导页
    └── [已完成 KYB] → 跳转至快速开始页面/仪表盘
    ↓
[验证失败] → 显示错误提示（账号或密码错误）
```

### 3.3 安全措施
- 连续 5 次登录失败，锁定账号 15 分钟
- 登录成功后记录登录日志（IP、时间、设备信息）
- Session 过期时间可配置（默认 2 小时）
- 支持「记住我」延长会话有效期

### 3.4 忘记密码
- 通过注册邮箱发送密码重置链接
- 重置链接有效期 30 分钟
- 重置成功后强制重新登录

## 4. KYB 引导认证

### 4.1 触发条件
- 商户注册成功后首次登录
- 或商户 KYB 认证状态为「未完成」时访问任何功能页面

### 4.2 KYB 认证流程

```
登录成功，检测到 KYB 未完成
    ↓
展示 KYB 引导页面
    ↓
填写公司基本信息
  - 公司注册地
  - 公司注册号
  - 营业执照号
  - 公司类型（有限公司/合伙企业等）
    ↓
填写法人/实控人信息
  - 姓名、国籍、证件类型及号码
  - 持股比例
    ↓
上传合规文件
  - 营业执照副本
  - 法人/实控人身份证件
  - 公司章程（如适用）
  - 银行开户证明
    ↓
提交 KYB 认证申请
    ↓
进入审核状态
  ├── [审核通过] → 通知商户，解锁全部功能
  ├── [审核拒绝] → 通知商户，说明拒绝原因，支持重新提交
  └── [需要补充材料] → 通知商户，指明需要补充的内容
```

### 4.3 KYB 认证状态

| 状态 | 说明 | 用户可执行操作 |
|------|------|----------------|
| 未开始 | 尚未提交 KYB 信息 | 填写并提交 KYB 信息 |
| 审核中 | 已提交，等待审核 | 查看提交的信息（只读） |
| 已通过 | 审核通过 | 正常使用所有功能 |
| 已拒绝 | 审核未通过 | 查看拒绝原因，修改后重新提交 |
| 需补充材料 | 审核方要求补充 | 上传补充材料后重新提交 |

### 4.4 KYB 与入驻申请的关系
- KYB 认证是商户身份合规验证，由 OSLPay 平台审核
- 入驻申请（Onboarding）是商户业务信息的提交，用于开通支付能力
- 流程顺序：注册 → KYB 认证 → 入驻申请
- KYB 未通过前，入驻申请功能受限（沙箱环境可跳过 KYB）

## 6. 验收标准

- [ ] 商户可通过邮箱完成注册，收到验证邮件并完成验证
- [ ] 注册后可使用邮箱和密码登录
- [ ] 首次登录自动跳转至 KYB 引导页
- [ ] KYB 认证未完成时，访问其他功能页面被拦截并引导至 KYB 页面
- [ ] KYB 提交后状态正确流转（审核中 → 通过/拒绝）
- [ ] 审核拒绝后可查看原因并重新提交
- [ ] 忘记密码流程正常工作
- [ ] 连续登录失败触发账号锁定
