# 开发者控制台与环境管理

## 1. 功能概述

开发者控制台是技术人员的操作中枢，提供 API 凭证查看、全局环境切换、沙箱测试指引和功能快速导航。

**适用角色**: 管理员、技术人员

## 2. API 凭证管理

展示当前商户的关键凭证信息：

| 凭证项 | 说明 | 操作 |
|--------|------|------|
| App ID | 应用唯一标识符 | 复制 |
| API 端点 | API 服务器地址（随环境切换） | 复制 |
| API 公钥 | 用于验证签名的 RSA 公钥 | 复制 |
| Webhook 公钥 | 用于验证 Webhook 推送 + 加密敏感字段 | 复制 |

- 点击「复制」按钮一键复制到剪贴板
- 端点地址随全局环境切换自动变更：
  - 生产: `https://openapi.osl-pay.com`
  - 沙箱: `https://openapitest.osl-pay.com`

## 3. 全局环境切换

### 3.1 切换入口
- 位于顶部栏右上角
- 显示当前环境标识：「生产」（红色背景）/ 「沙箱」（绿色背景）
- 点击即可切换

### 3.2 切换影响范围
环境切换为全局操作，影响以下所有功能模块：

| 模块 | 影响内容 |
|------|----------|
| API 凭证 | 端点地址切换 |
| API 文档 | 在线试用请求发送到对应环境端点 |
| 入驻申请 | 生产 4 步 vs 沙箱 3 步流程 |
| 签名工具 | 无直接影响（工具独立） |
| Webhook | 配置和日志按环境隔离 |
| 仪表盘 | 展示对应环境的交易数据 |
| MCP 配置 | 生成对应环境的 MCP 配置 |

### 3.3 数据隔离
- 生产环境和沙箱环境的数据完全独立
- 切换环境不会导致数据丢失
- 各环境有独立的订单、Webhook 配置、API 日志

## 4. 沙箱测试指南

当环境切换到沙箱时，开发者控制台展示沙箱测试指南：

### 4.1 测试卡号

| 卡类型 | 卡号 | 行为 |
|--------|------|------|
| Visa 3DS Frictionless | 4242 4242 4242 4242 | 自动通过 3DS 验证 |
| Visa 3DS Challenge | 4539 3732 9896 7400 | 需用户完成 3DS 挑战 |
| Visa 余额不足 | 4532 2274 1657 1592 | 支付失败 |

- 有效期：未来 20 年内任意日期
- CVV：任意 3 位数字

### 4.2 支持的货币和网络

| 法币 → 加密货币 | 最小金额 | 最大金额 |
|-----------------|----------|----------|
| EUR → USDT/USDC | 1 EUR | 43,000 EUR |
| EUR → ETH | 1 EUR | 43,000 EUR |
| USD → USDT/USDC | 1 USD | 53,000 USD |

**支持的网络**: ERC20, TRC20, BEP20, Polygon, Arbitrum, Optimism, Solana 等

### 4.3 沙箱限额
- 单笔交易：$10,000
- 日累计：$50,000
