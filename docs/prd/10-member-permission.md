# 成员与权限管理

## 1. 功能概述

成员与权限管理模块允许管理员管理商户团队成员，通过邮箱邀请新成员并分配角色，控制不同成员的功能访问范围。

**适用角色**: 仅管理员

## 2. 成员列表

### 2.1 展示信息

| 字段 | 说明 |
|------|------|
| 成员姓名 | 用户姓名 |
| 邮箱 | 登录邮箱 |
| 角色 | 管理员 / 业务人员 / 技术开发 |
| 状态 | 已激活 / 待接受 |
| 加入时间 | 首次加入或邀请时间 |

### 2.2 角色权限说明
页面顶部展示三种角色的权限概要：
- **管理员**: 全部权限
- **业务人员**: 仅看申请/账单
- **技术开发**: 仅看开发者套件

## 3. 邀请新成员

### 3.1 操作流程
1. 点击「邀请成员」按钮
2. 弹出模态框，填写：
   - 邮箱地址（必填）
   - 角色权限（下拉选择）
3. 点击「发送邀请」
4. 系统发送邀请邮件
5. 成员列表显示新成员，状态为「待接受」

### 3.2 邀请规则
- 邀请邮件有效期 7 天
- 超时未接受需重新邀请
- 同一邮箱不可重复邀请
- 被邀请人通过邮件链接完成注册后状态变为「已激活」

## 4. 移除成员

### 4.1 操作流程
1. 在成员列表找到目标成员
2. 点击「移除」按钮
3. 确认对话框二次确认
4. 确认后成员从列表移除

### 4.2 移除规则
- 无法移除自己的账号
- 移除后该成员立即无法访问平台
- 移除操作不可撤销（需重新邀请）
- 移除操作记录审计日志

## 5. 验收标准

- [ ] 仅管理员角色可看到成员管理菜单
- [ ] 成员列表展示姓名、邮箱、角色、状态、加入时间
- [ ] 可通过邮箱邀请新成员并分配角色
- [ ] 邀请后系统发送邀请邮件，列表显示「待接受」
- [ ] 可移除成员，移除前需二次确认
- [ ] 无法移除自己的账号
- [ ] 邀请邮件 7 天过期
