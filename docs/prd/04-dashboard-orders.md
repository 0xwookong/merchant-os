# 业务数据仪表盘与订单管理

## 1. 功能概述

仪表盘是商户业务运营的核心视图，提供关键业务指标概览、交易趋势图表和订单管理能力，帮助业务人员实时了解运营状况。

**适用角色**: 管理员、业务人员

## 2. 关键指标概览

顶部展示 4 个核心指标卡片：

| 指标 | 说明 | 展示格式 |
|------|------|----------|
| 交易总额 | 累计交易金额 | $XXX,XXX.XX + 环比变化率 |
| 成功率 | 交易成功百分比 | XX.X% + 环比变化率 |
| 订单数量 | 总订单数 | X,XXX + 环比变化率 |
| 活跃用户数 | 活跃终端用户数 | X,XXX + 环比变化率 |

### 指标卡片交互
- Hover 时卡片升起（shadow-md）
- 环比变化率：上升显示绿色箭头，下降显示红色箭头
- 支持按时间范围筛选，所有指标联动更新

## 3. 时间范围筛选

| 选项 | 说明 |
|------|------|
| 今日 | 当天 00:00 至当前时间 |
| 近 7 天 | 最近 7 个自然日 |
| 近 30 天 | 最近 30 个自然日 |
| 自定义 | 用户选择开始和结束日期 |

切换时间范围后，所有图表和数据联动刷新。

## 4. 订单列表

### 4.1 列表字段

| 列名 | 说明 |
|------|------|
| 订单号 | 唯一标识，如 ORD20240303001 |
| 金额 | 法币金额 + 币种（如 $100.00 USD） |
| 加密货币 | 目标加密货币及数量（如 0.05 ETH） |
| 支付方式 | CARD / GOOGLEPAY / APPLE PAY |
| 状态 | CREATED / PROCESSING / SUCCESSED / COMPLETED / FAILED |
| 创建时间 | 订单创建时间 |

### 4.2 筛选条件

| 筛选项 | 选项 |
|--------|------|
| 状态 | 全部 / CREATED / PROCESSING / SUCCESSED / COMPLETED / FAILED |
| 支付方式 | 全部 / CARD / GOOGLEPAY / APPLE PAY |
| 时间范围 | 与顶部时间筛选联动 |

### 4.3 分页
- 每页显示 20 条记录
- 提供「上一页」「下一页」翻页按钮
- 显示当前页码和总页数

## 5. 订单详情

点击订单行展开或进入详情页，包含：

### 5.1 基本信息
- 订单号、金额、支付方式、创建时间
- 法币信息（币种、金额）
- 加密货币信息（币种、数量、目标网络）

### 5.2 状态流转历史

```
CREATED → PROCESSING → SUCCESSED → COMPLETED
                    └→ FAILED
```

时间线展示每个状态的变更时间。

### 5.3 链上信息
- 交易哈希（TX Hash）
- 链上确认数
- 区块高度
- 链上浏览器链接

## 6. 数据导出

- 点击「导出 CSV」按钮
- 根据当前筛选条件（状态、支付方式、时间范围）导出
- CSV 文件包含所有订单列表字段
- 支持 Excel 打开

## 7. 订单状态说明

| 状态 | 中文 | 含义 | 标签颜色 |
|------|------|------|----------|
| CREATED | 已创建 | 订单已创建，等待支付 | 灰色 |
| PROCESSING | 处理中 | 支付处理中 | 蓝色 |
| SUCCESSED | 支付成功 | 法币支付成功，等待加密货币到账 | 黄色 |
| COMPLETED | 已完成 | 加密货币已到账，订单完成 | 绿色 |
| FAILED | 失败 | 支付失败或订单异常 | 红色 |
