# 域名白名单管理

## 1. 功能概述

域名白名单管理模块用于配置允许调用 Web SDK 的业务域名。只有白名单中的域名才能正常调用 OSLPay Web SDK，实现来源校验，防止未授权的域名滥用 SDK。

**适用角色**: 管理员、技术人员

## 2. 功能说明

### 2.1 查看域名列表

| 字段 | 说明 |
|------|------|
| 域名地址 | 包含协议的完整域名 |
| 状态 | 已激活 |
| 添加时间 | 域名添加时间 |

### 2.2 添加域名

**操作步骤**:
1. 在输入框中输入完整域名（含协议）
2. 点击「添加」按钮
3. 系统验证域名格式
4. 验证通过后加入列表

**域名格式要求**:
- 必须包含协议: `http://` 或 `https://`
- 支持子域名: `https://app.example.com`
- 支持端口号: `https://localhost:3000`
- 不支持通配符: 不能使用 `*.example.com`

**合法示例**:
- `https://www.example.com`
- `https://app.example.com`
- `http://localhost:3000`
- `http://127.0.0.1:8080`

**错误处理**:
- 域名格式无效: 显示格式错误提示
- 域名已存在: 显示重复提示
- 不是有效 URL: 显示无效 URL 提示

### 2.3 移除域名
1. 点击域名行右侧「移除」按钮
2. 确认对话框二次确认
3. 确认后从列表移除

### 2.4 用途说明
页面顶部信息提示框说明白名单机制：
- 只有白名单中的域名才能调用 Web SDK
- 请添加业务域名（包含协议）
- 支持子域名和端口号
